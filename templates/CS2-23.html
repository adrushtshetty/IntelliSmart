Problem 2/23

Determine the scalar components Ra and Rb of the force R along the nonrectangular axes a and b. Also determine the orthogonal projections Pa of R onto axis a.

Answer:

Form a parallelogram using the Pralallelogram Law

From Sine Rule
Let x = 180 - (α+β)
R / sin(x) = Fa / sin(β) = Fb / sin(α)
R / sin({{rt_x}}°) = Fa / sin({{rt_b}}°) = Fb / sin({{rt_a}}°)

Fa = R * sin(β) / sin(x)
Fa = {{rt_r}}*sin({{rt_b}}°) / sin({{rt_x}}°)
Fa = {{rt_fa}} N

Fb = R * sin(α) / sin(x)
Fb = {{rt_r}}*sin({{rt_a}}°) / sin({{rt_x}}°)
Fb = {{rt_fb}} N

Component of R on axis a:
Pa = R * cos(α)
Pa = {{rt_r}} * cos({{rt_a}}°)
Pa = {{rt_pa}} N
