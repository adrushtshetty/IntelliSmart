Problem 6/3

The force P is applied to the 50-kg block when it is at rest. Determine the magnitude and direction of the friction force exerted by the surface on the block if (a) P = 0, (b) P = 200 N, and (c) P = 250 N. (d) What value of P is required to initiate motion up the incline? The coefficients of static and kinetic friction between the block and the incline are μs. = 0.25 and μk = 0.20, respectively

Answer:

Given:
θ = 15
α = 20
μs = 0.25
μk = 0.20

a)
P=0 [Given]
N = | mgcosθ-Psinα |
N = 473.6248 N

Ft = | mgsin15 - Pcos20 | [Tractive Force]
Ft = 126.9073 N

Ff = N x μs [Static Friction]
Ff = 118.4062 N


Since Ff < Ft,
Ff = N x μk [Kinetic Friction]
Ff = 94.72496 N
Direction of Ff is up the incline



b)
P=200 N [Given]
N = | mgcosθ-Psinα |
N = 405.2207 N

Ft = | mgsin15 - Pcos20 | [Tractive Force]
Ft = 61.0311 N

Ff = N x μs [Static Friction]
Ff = 101.3051 N


Since Ff > Ft,
Ff = Ft
Ff = 61.0311 N
Direction of Ff is up the incline



c)
P=250 N [Given]
N = | mgcosθ-Psinα |
N = 388.1197 N

Ft = | mgsin15 - Pcos20 | [Tractive Force]
Ft = 108.0156 N

Ff = N x μs [Static Friction]
Ff = 97.0299 N


Since Ff < Ft,
Ff = N x μk [Kinetic Friction]
Ff = 77.6239 N
Direction of Ff is down the incline



d)
N = | mgcosθ-Psinα |
Ft = Pcos20 - mgsin15 [Tractive Force]
Fmax = N x μs [Static Friction]
Fmax = Ft
N x μs = Pcos20 - mgsin15
P=mg(μs.cos15 + sin15)/(cos20 + μs.sin20)
P=239.3659 N
